Steps for compiling and running:

STEP 1:
Downloading libraries:
1. Download nltk:
   Command: python3 -m nltk.downloader 'punkt'

STEP 2:
Implementing Parallel Data:
1. Download the parallel corpus.
2. Change the directory path inside the main.py code.

STEP 3:
Compiling:
1. Include the library path while compiling.
2. Compile and run the main.py file.

Steps for calculating cosine similarity and jaccard coefficient:
1. Copy the two documents inside the same directory.
2. Run JaccardCosine.java and give the names of files as inputs.
